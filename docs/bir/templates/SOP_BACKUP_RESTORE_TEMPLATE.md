# SOP Template: Backup and Restore

## Objective
Define periodic backup, retention, restore testing, and approval process.

## Scope
Covers CAS database and generated accounting reports.

## Procedure
1. Execute backup via CAS backup module.
2. Record backup reference, date/time, and operator.
3. Store backup file in controlled storage location.
4. Perform periodic restore drill in non-production environment.
5. Document result and sign-off.
