# SOP Template: Change Management

## Objective
Ensure all CAS changes are approved, tested, and documented.

## Procedure
1. Raise change request with reason and impact.
2. Review and approve by designated approvers.
3. Implement in controlled environment.
4. Execute regression/UAT tests.
5. Deploy with release notes and version update.
6. Archive evidence and approvals.

## Required Artifacts
- Change request form
- Test evidence
- Approval log
- Release notes
