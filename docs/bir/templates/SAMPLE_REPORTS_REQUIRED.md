# Sample Reports Required for CAS Registration Package

Prepare representative outputs for the following:

1. Trial Balance (monthly, quarterly, annual)
2. Balance Sheet
3. Income Statement
4. Journal Book
5. General Ledger Book
6. Sales Journal
7. Purchase Journal
8. Cash Receipts Journal
9. Cash Disbursements Journal
10. Subsidiary Ledgers (AR/AP/Customer/Supplier)

For each report include:
- Date range used
- PDF and Excel export copy
- Report reference number
- Page count
- Header with company TIN and address
